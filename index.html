Summoners War Speed Tuning Calculator

Select the monsters you'd like to tune using the selectors below (type the name once you click on the dropdown for quick search).

The tuning order is left to right. (On mobile top to bottom)

Speed from runes is the green + number you see next to speed in your monster details.

*Note: The latest SWARFARM data is not accurate. Some of the monster information might not be correct. Double-check the monster details before using the calculator.
